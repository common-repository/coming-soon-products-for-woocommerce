== Description ==

Trigger your unavailable products in coming soon mode and let your customers know when it will be available.
Coming Soon Products for WooCommerce plugin offers the option to enable "Coming Soon" mode for any product.
You can set the arrival time of each product and display a countdown timer until the arrival time comes.


== Features ==

* Enable | Disable Coming Soon Mode for each product

* Add Coming Soon Text for each product.

* Display countdown counter for each coming soon product.

* Hide the price of each coming soon product.

* Show a coming soon badge over each coming soon product in shop pages.

* Get full control over the style of the countdown counter and the coming soon badge.

* Track coming soon products from the admin top bar.

* Use "[wcsamm-soon-products]" shortcode to get list of coming soon products. The shortcode follows the same attributes of the WooCommerce shortcodes.

= How to use? =
The main settings for coming soon badge and countdown counter are located at WooCommerce->Settings->Coming Soon Products Tab. coming soon settings for each product are located at "Coming Soon" Tab in product data section inside product edit page.
